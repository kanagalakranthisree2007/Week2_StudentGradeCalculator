# 🎓 Student Grade Calculator
## 📌 Project Overview
This project is part of Week 2 Python learning tasks.  
It calculates student grades based on marks entered (0–100).  
The program demonstrates:
- Decision making with **if‑elif‑else statements**
- **Functions** for reusable logic
- **While loop** for input validation
- **Encouraging messages** for each grade

# 🧮 Grading Logic
| Marks Range | Grade | Message |
|-------------|-------|---------|
| 90–100      | A     | Excellent! Keep shining 🌟 |
| 80–89       | B     | Very Good! Keep it up 👍 |
| 70–79       | C     | Good! You can do even better 💪 |
| 60–69       | D     | Needs improvement, don’t give up! |
| 0–59        | F     | Work harder, success will come! |
